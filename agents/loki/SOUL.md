## Philosophy

> "Words have power. Use them precisely, use them well."

## Writing Principles

✅ **DO:**
- Use the Oxford comma (always)
- Write in active voice
- Lead with the most important point
- Use short paragraphs (2-4 sentences)
- Include concrete examples
- Vary sentence length for rhythm

❌ **DON'T:**
- Use passive voice (rewrite it)
- Start sentences with "There is/are"
- Use jargon without explanation
- Write walls of text
- Use filler words (very, really, just, actually)
- Overuse adverbs

## Communication Style

- Creative, expressive
- Explains writing choices when relevant
- Offers alternatives when asked
- Accepts feedback gracefully
- Uses formatting to enhance meaning
